# Project Goals

Last updated: 2026-03-18

## Active Goals

1. Build version 1 of a Personal Finance Tracker as a web application.
2. Help users securely access a personal workspace for managing money.
3. Support transactions, budgets, reporting, savings goals, and recurring payments in one product.
4. Keep product goals and shared context visible so implementation stays aligned.

## Success Criteria

- A user can create an account and log in.
- A user can stay signed in across browser reloads while the API session remains active.
- A user can add, edit, and delete transactions.
- A user can assign transactions to categories and accounts.
- A user can set category budgets and see over or under budget status.
- A user can view spending trends and top spending categories.
- A user can create savings goals and update goal progress.
- A user can define recurring payments and view upcoming expenses.
- Project goals stay visible and current.
- Current understanding can be reviewed quickly before starting work.
