# Project Goals

Last updated: 2026-03-16

## Active Goals

1. Build version 1 of a Personal Finance Tracker as a web application.
2. Help users record income and expenses quickly and understand where their money is going.
3. Support monthly budgets, savings goals, recurring payments, and basic financial trends.
4. Keep product goals and shared context visible so implementation stays aligned.

## Success Criteria

- A user can create an account and log in.
- A user can add a transaction in under 15 seconds.
- A user can view current month spending by category.
- A user can compare budget versus actual spending.
- A user can identify recurring payments and upcoming bills.
- A user can view simple trend charts over time.
- Project goals stay visible and current.
- Current understanding can be reviewed quickly before starting work.
